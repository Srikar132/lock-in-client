## ❓ Frequently Asked Questions

### Can I really not bypass the app blocking?
With **Strict Mode** enabled, no—that's the point! Regular mode allows you to stop sessions early, but Strict Mode is unbreakable until the timer ends. This helps build real discipline.

### What happens in an emergency?
Emergency calls and messages always work. LOCK-IN never blocks essential phone functions—only the apps you choose.

### Will this drain my battery?
No! LOCK-IN uses minimal background resources. Most users report no noticeable battery impact.

### Can I use it for work apps too?
Absolutely! Block social media during work hours, or block work apps during personal time. Total flexibility.

### What if I need to use a blocked app for something important?
Use **Classic Mode** (not Strict Mode) so you can stop the session if needed. Or use the break time in Pomodoro mode.

### Do I need to be online?
LOCK-IN works offline for focus sessions. Sync and groups require internet connection.

### How is this different from Digital Wellbeing?
LOCK-IN is **actively protective** rather than passive. It blocks apps during sessions, has strict modes, includes AI assistant, group features, and gamification—all designed for serious focus.
